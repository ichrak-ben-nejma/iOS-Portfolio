# Awisso+ (Client Project)

**Role:** iOS Developer

## Description
Awisso+ is a business analytics iOS application designed for online store owners.  
It provides insightful sales and order statistics, helping businesses track performance and manage customer orders efficiently.

## Key Features
- Sales and orders analytics dashboard
- Accept or cancel customer orders
- Interactive charts for statistical insights
- Custom calendar for order tracking
- Real-time customer support via live chat
- Smooth and modern SwiftUI-based UI

## Technologies & Tools
- SwiftUI
- MVVM Architecture
- Alamofire
- UserDefaults
- Custom Charts Library
- Custom Calendar Component
- Lottie Animations
- Customerly (Live Chat SDK)

## My Contribution
- Implemented SwiftUI user interfaces based on provided designs
- Integrated backend REST APIs for orders, analytics, and statistics
- Built custom chart components for data visualization
- Managed app state using MVVM architecture
- Added animations and live chat support
- Ensured smooth user experience and performance optimization
